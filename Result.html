實驗結果與分析¶

實驗流程說明¶

我們的分析方法如下圖:

  1. 首先探討與制定控制變因
  2. 採用ANOVA分析, 針對實驗組與對照組進行分析. 我們採用P-value為0.05為標準, 小於此標準為顯著.
  3. 如果ANOVA分析結果不顯著, 則重新探討與制定控制變因 (回到第1點).
  4. 若ANOVA分析結果顯著, 我們再判斷控制變因是否為連續型資料. 若為連續型資料, 則採用迴歸分析繼續分析控制變因, 反之則採用箱型圖分析.

ANOVA分析介紹¶

ANOVA分析的精神就是最大化組間變異與最小化組內變異. 最大化組間變異目的是希望實驗的控制變因是有顯著的, 最小化組內變異的目的是希望實驗水準夠高, 量測誤差很小. 因此ANOVA分析制定: $F=\frac{MS_B}{MS_W}$, 其中${MS_B}$為組間均方和, 我們期望越大越好, 代表控制變因顯著. ${MS_W}$為組內均方和, 越小越好, 代表量測誤差小. 最後透過F查表P-Value, 我們設定0.05為顯著水準.

df = df0[['原音量', '非牛頓_薄', '非牛頓_厚']]

# 計算 ANOVA 表格的各個組成部分
# 群組數量：DataFrame 的欄數
n_groups = len(df.columns)
# 總觀測數量：每個群組的樣本數乘以群組數量
n_total = len(df) * n_groups

# 計算總平均值 (grand_mean) 和每個群組的平均值
grand_mean = np.mean(df.values)  # 所有觀測值的平均值
group_means = df.mean()  # 各群組的平均值

# 計算平方和 (Sum of Squares, SS)
# 組間平方和 (SSB)：反映各群組平均值與總平均值之間的差異
SSB = sum(len(df) * (mean - grand_mean) ** 2 for mean in group_means)

# 組內平方和 (SSW)：反映群組內部數據的變異程度
SSW = sum(sum((df[col] - group_means[col]) ** 2) for col in df.columns)

# 總平方和 (SST)：組間平方和與組內平方和之和
SST = SSB + SSW

# 計算自由度
# 組間自由度：群組數量減 1
df_between = n_groups - 1
# 組內自由度：總觀測數量減去群組數量
df_within = n_total - n_groups
# 總自由度：總觀測數量減去 1
df_total = n_total - 1

# 計算平方均值 (Mean Squares, MS)
# 組間平方均值：組間平方和除以組間自由度
MSB = SSB / df_between
# 組內平方均值：組內平方和除以組內自由度
MSW = SSW / df_within

# 計算 F 統計量
# F 值 = 組間平方均值 / 組內平方均值
F = MSB / MSW

# 計算對應的 p 值
# 使用 scipy.stats.f.sf 計算 F 分布的右尾概率，即 p 值
p_value = f.sf(F, df_between, df_within)

#可以直接套用f_oneway函數計算每個群組之間的F統計量與對應的p值,用來檢視F與p_value是否算錯
#f_stat, p_value_anova = f_oneway(df["Group1"], df["Group2"], df["Group3"], df["Group4"])

# 建立 ANOVA 表格，整合計算結果方便查看
anova_table = pd.DataFrame({
    "Source": ["Between Groups", "Within Groups", "Total"],
    "SS": [SSB, SSW, SST],
    "df": [df_between, df_within, df_total],
    "MS": [MSB, MSW, None],
    "F": [F, None, None],
    "p-value": [p_value, None, None]
})

# 顯示 ANOVA 表格
print(anova_table)

我們採用迴歸分析, 來驗證ANOVA分析不顯著的結果. 結果顯示: R-square為0.0001, P-value=0.9125. 顯示打洞數量沒有顯著影響隔音效果.

dfA = pd.read_excel('data.xlsx', sheet_name='打洞數量分析')

# 取出 A 和 B 欄位
X = dfA['打洞數'].values  # 自變數
Y = dfA['音量'].values  # 因變數

# 執行線性回歸
slope, intercept, r_value, p_value, std_err = linregress(X, Y)

# 輸出回歸結果
print(f"迴歸方程式: B = {slope:.4f} * A + {intercept:.4f}")
print(f"R² 值: {r_value**2:.4f}")
print(f"P 值: {p_value:.4e}")
print(f"標準誤差: {std_err:.4f}")

# 繪製散佈圖與回歸線
plt.figure(figsize=(12, 8))
plt.scatter(X, Y, label='Data Points')
plt.plot(X, slope * X + intercept, color='red', label='Regression Line')
plt.xlabel('Number of Holes', fontsize=16)
plt.ylabel("Volume", fontsize=16)
plt.xticks(fontsize=14)
plt.yticks(fontsize=14)
plt.legend()
plt.show()
